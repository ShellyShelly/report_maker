Description Of Web-Site

How everything is arranged here.

To make your reports, you must add some copulas in an inset(Connections), and, after that you can to go across in an inset (Reports). 1 copulas = 1 project. 

Copula.

A copula it data for the search of tweets/retweets, posts/reposts for one specific project. A copula consists of Hashtag, Day of week, Type of report, Link to twitter of this project, Number in the spreadsheet for twitter, ID of facebook of this project, Number in the spreadsheet for facebook. In a copula it is possible that you can add twitter + facebook at the same time, and of course you can do it separately for twitter only, or only for facebook.

ID for facebook you can get here https://findmyfbid.com/ , where you must specify a link to the facebook project page.

Hashtag.

Hashtag is compulsory at filling. It use as name of copula. At creation

 of report, a text file will be named after like a hashtag. Even if the project does not need to create posts, tweets, the hashtag will be used as a landmark. 

***Fill the field of Hashtag is needed without # , just enter only that after this sign. For example: In the project BAANX is need use Hashtag #BAANXICO, so you must Fill the field of Hashtag like this "BAANXICO"***

Day of week.

It is a day of sending of report to this project. Specifying one time in a copula the day of sending of report, you do not need to do it each time, when need to hand over a report. 

For example: you need to hand over a report for "Sunday", so you open inset Reports, choose "Sunday"(date is under this day) and the system automatically generates all reports in which the reporting day is Sunday.

Number in the spreadsheet.

In some projects there are such requirements to the reports, how to Specify the number in a spreadsheet. Specifying is possible as well as for twitter, and for to the facebook. If you specified a number, so, then, when you create a report, it will be automatically added.

This field is not obligatory.

Type of report.

Usual, Numbered, Full.

Full – when the tweets/retweets/posts/reposts are numbered and their date is set.

Example:

 

              1 – 23.06: https://twitter.com/***

              2 – 22.06: https://twitter.com/***

              3 – 21.06: https://twitter.com/***

Numbered – only numeration of links

  1. 1.https://twitter.com/***

Usual – only links

                  https://twitter.com/***
